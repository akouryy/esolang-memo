## 2文字→1文字

* `$`
  * `$a` → `A`
* `^`
  * `^i` → `I`
* `c`
  * `c$` → `C`
  * `cc` → `S`

* `d`
  * `d$` → `D`
  * `dd` → `J` (空行)
  * `dh` → `X`
  * `dl` → `x`
* `g`
  * `gg` → `H` (スクロール上端)
* `j`
  * `j^` → `+`
* `k`
  * `k^` → `-`

* `y`
  * `yy` → `Y`
